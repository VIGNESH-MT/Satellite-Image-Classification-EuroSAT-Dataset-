About EuroSAT Classifier

Advanced satellite image classification using deep learning and Google Earth Engine

Project Overview

The EuroSAT Satellite Image Classifier is a comprehensive machine learning project that combines state-of-the-art deep learning models with Google Earth Engine's powerful geospatial analysis capabilities to classify satellite images into different land use categories.

This project demonstrates the practical application of transfer learning, computer vision, and geospatial analysis in solving real-world environmental monitoring and land use classification challenges.

Deep Learning Models

VGG16 Architecture
  • Pre-trained on ImageNet
  • Transfer learning with frozen base layers
  • Custom classification head
  • Optimized for speed and accuracy
ResNet50 Architecture
  • Residual connections for deeper networks
  • Better gradient flow
  • Higher accuracy for complex patterns
  • Fine-tuning capabilities
Training Features
  • Data augmentation
  • Early stopping
  • Learning rate scheduling
  • Model checkpointing

How to Use

Step 1: Upload Image

Select or drag-and-drop a satellite image in JPG, PNG, or TIFF format. Maximum file size is 16MB.

Step 2: Choose Model

Select between VGG16 (faster) or ResNet50 (more accurate) based on your needs.

Step 3: View Results

Analyze the classification results, confidence scores, and probability distributions.

Step 4: Explore Maps

Use the interactive map to explore the location and validate results with real satellite data.

Step 5: Analyze Location

Click on the map or enter coordinates to get Google Earth Engine analysis for any location.

Tips for Best Results
  • Use high-quality satellite images
  • Ensure images show clear land use patterns
  • Try both models for comparison

API Endpoints

Available Endpoints:
  • GET /api/model_info - Get information about loaded models
  • POST /api/analyze_coordinates - Analyze specific coordinates with GEE
